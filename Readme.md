## grpc-contacts-client

This backend server acts as a server point for frontend requests for purposes of the coding challenge.

To run locally:
1. Run `npm install`
2. Run `npm run dev`

The default predefined user accounts for login are:
```
[
    {
     "username":"admin",
     "password":"admin"
    },
    
    {
     "username":"staff",
     "password":"staff"
    }
]
```
